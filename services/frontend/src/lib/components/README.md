# Reusable Svelte Components

This directory contains reusable UI components used across multiple pages.

## Planned Components

- `PollCard.svelte` - Display poll summary in grid view
- `VoteForm.svelte` - Poll voting interface
- `ResultsChart.svelte` - Real-time results visualization

## Usage

Import components from this directory:
```svelte
<script>
  import PollCard from '$lib/components/PollCard.svelte';
</script>
```
